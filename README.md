# 斷易
使用 Unity 跨平台開發的 2D 遊戲專案，未來遊戲推出時僅支援 Windows 平台，應該。

## 結構簡介

- `Assets/`：放置所有遊戲內容，包括腳本、圖片、Prefab 等。
- `ProjectSettings/`：Unity 的設定檔，建議版本控制。
- `Packages/`：套件資訊，建議保留於 Git 以確保跨機使用一致性。
